# 🌐 IP Address & Network Info Lookup Tool

A utility tool for CS students to look up IP details and understand networking basics.  
Built with **HTML + CSS + JavaScript** using the free API `ipapi.co`.

---

## ✨ Features
- 🔍 Fetch current public IP
- 🌍 Show country and country code
- 🏢 Display ISP/organization
- ⏰ Show timezone
- 🎨 Clean card-style responsive UI
