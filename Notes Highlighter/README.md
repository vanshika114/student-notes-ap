# 📝 Notes Color Highlighter

A simple **Notes App** with color highlighting built using **HTML + CSS + JavaScript**.  
Add notes, highlight them in different colors (yellow, pink, green, blue), and persist your choices with `localStorage`.

---

## ✨ Features
- ➕ Add notes with text
- 🎨 Highlight notes in **yellow, pink, green, or blue**
- 🔄 Quick toggle buttons for changing colors
- 💾 Persistent storage using `localStorage`
- 🎨 Modern responsive UI
