# 📔 Daily Journal / Mood Tracker

A simple **Daily Journal & Mood Tracker** built with **HTML + CSS + JavaScript**.  
Log your mood with emojis, write a short entry, and view past logs.

---

## ✨ Features
- 😄 Mood selection (emoji picker)
- 🖊️ Journal entry input
- 💾 Save entries with localStorage
- 📅 Past entries list view
