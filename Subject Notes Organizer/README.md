# 📚 Subject-wise Notes Organizer

A simple **Notes Organizer App** built with **HTML + CSS + JavaScript**.  
Organize notes by subject (Maths, Science, CS, History), filter by category, search quickly, and persist data with `localStorage`.

---

## ✨ Features
- ➕ Add notes with subject tags
- 📂 Organize notes by subject (Maths, Science, CS, History)
- 🔍 Quick search functionality
- 🗂️ Category filtering
- 💾 Persistent storage with localStorage
- 🎨 Modern responsive UI
