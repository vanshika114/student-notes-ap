# 🧠 True or False Quiz Game

An interactive **True or False Quiz Game** built with **HTML + CSS + JavaScript**.  
Test your knowledge across multiple subjects with a timer, score tracking, animations, and a final results screen.

---

## ✨ Features
- 📚 Multiple subjects (Science, History, Sports, etc.)
- ⏳ Countdown timer per question
- 🏆 Score tracking with instant feedback
- 🎬 Smooth animations for transitions
- 🎉 Final results screen with restart option
- 🎨 Modern responsive UI
