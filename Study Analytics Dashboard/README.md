# 📊 Study Time Analytics Dashboard

An interactive **Study Analytics Dashboard** built with **HTML + CSS + JavaScript**.  
Visualize weekly study hours, subject breakdown, consistency streaks, and daily study heatmaps.

---

## ✨ Features
- 📈 Weekly study hours line chart
- 🥧 Subject breakdown pie chart
- 🔥 Consistency streak tracker
- 🗓️ Daily study heatmap visualization
- 💾 Persistent streak tracking with localStorage
- 🎨 Modern responsive UI
