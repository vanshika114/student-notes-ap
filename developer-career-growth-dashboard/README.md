# Developer Career Growth Dashboard 🚀💻

A comprehensive personal career growth dashboard designed for aspiring software engineers and tech professionals. Track your skill progression, catalog open-source and personal contributions, map out learning paths, estimate resume readiness, and log DSA solutions from a single glassmorphic interface.

## 🌟 Features

1. **Profile Summary & XP Rank**:
   - Live Level-up and rank titles based on XP earned through study and contributions.
2. **Interactive Skills Progression Matrix**:
   - Expandable skill nodes mapping Frontend, Backend, DevOps, and DSA. Level up skills from Novice to Expert.
3. **Contribution Tracker (CRUD)**:
   - Log and edit your PRs, Issues, and reviews with live count meters.
4. **Learning Roadmaps Hub**:
   - Pre-populated developer roadmap resources with quick completion toggles.
5. **Resume Score Estimator**:
   - Checkoff criteria to dynamically calculate your resume readiness percentage.
6. **DSA Coding Logs**:
   - Record solved coding challenges with difficulty tags.

## 🛠 Setup

To run locally, start a server or open `index.html` directly in your browser:
```bash
npm run dev
```
Navigate to `/developer-career-growth-dashboard/index.html`.
