# FOCUS — Task Manager

A clean, modern, single-page to-do app with a dark glassmorphism UI, aurora background animations, and smooth micro-interactions. Built with vanilla HTML, CSS, and JavaScript — no frameworks or dependencies.

---

## Project Structure

```
TO-DO-APP/

├── style.css    — All styles, animations, and theming
├── app.js       — Logic, state, localStorage, events
└── README.md    — This file
```
