# SNEK — Snake Game

A retro CRT-terminal Snake game with a matrix-green aesthetic, 10 speed levels, golden bonus food, and swipe support.

## Controls
- **Arrow keys / WASD** — change direction
- **Space** — start / pause / unpause
- **Swipe** — change direction on mobile
- **D-Pad buttons** — on-screen controls (mobile)

## Features
- 10 progressive speed levels (increases every 50 points)
- Red food (+10 × level), Gold bonus food (+50 × level, disappears after 5s)
- High score saved to localStorage
- Canvas pixel rendering with neon glow shadows
- CRT scanline + moving beam overlay
- Swipe gesture support

## Run
Open `index.html` in a browser — no setup needed.
