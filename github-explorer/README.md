# GitHub Explorer

A simple frontend project that fetches and displays public GitHub profile data, repositories, and contribution-style stats.

## Features
- Search any public GitHub username
- View profile details, stats, and avatar
- See top repositories sorted by stars
- Inspect language distribution and repo metrics

## Notes
- Uses GitHub's public REST API
- No authentication is required for basic usage
- Rate limits apply for unauthenticated requests

## Run
Open `index.html` in a browser or serve the folder with a local server.
