# 📊 Habit Tracker App

A simple and effective **Daily Habit Tracker** built with **HTML + CSS + JavaScript**.  
Track your habits, maintain streaks, and visualize weekly progress — all saved in your browser with `localStorage`.

---

## 🚀 Features
- ➕ Add custom habits
- ✅ Mark daily completion with a 7‑day grid
- 🔥 Streak counter for consecutive completions
- 📅 Weekly progress view
- 💾 Persistent data using `localStorage`
- 🗑️ Delete habits (optional enhancement)
