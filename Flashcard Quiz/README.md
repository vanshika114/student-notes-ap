# 📚 Flashcard Quiz App

An interactive **Flashcard Quiz App** built with **HTML + CSS + JavaScript**.  
Create custom flashcard decks, flip cards with animations, shuffle them, track your score, and export/import decks for reuse.

---

## ✨ Features
- ➕ Create custom flashcard decks
- 🔄 Front/back flip animation
- 🔀 Shuffle mode
- 🏆 Score tracking
- 📤 Export decks to JSON
- 📥 Import decks from JSON
- 💾 Persistent storage with localStorage
- 🎨 Modern responsive UI
