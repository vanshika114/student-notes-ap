# 📁 Contribution Structure

Each contributor should:

1. Create a new folder inside the repository.

Example:

```bash
/weather-card
/calculator
/navbar-component
```

2. Keep all project files inside that folder only.

Example:

```b
weather-card/
├── index.html
├── style.css
└── script.js
```

3. Do NOT modify other contributor folders unless asked.
