# ⏰ Digital Study Clock & Ambient Sounds

A immersive dashboard featuring a real-time digital clock synchronized with structural audio modules to provide focusing ambient background tracks (Rain, White Noise, Cafe) for long study sessions.

## 🚀 Features
- **Live Clock Synchronization:** Uses an active JavaScript interval engine mapping to standard browser localization clocks.
- **Integrated Ambient Tracks:** Native HTML5 `<audio>` streams configured to act as toggleable noise cancellation blocks.
- **Glassmorphic Design:** Styled with semi-transparent card backdrops, subtle hover transformations, and micro-animations.
