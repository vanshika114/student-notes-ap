# ✨ Motivational Quote Generator

A simple and elegant **Random Quote Generator** built with HTML, CSS, and JavaScript.  
Designed to keep students inspired while studying.

---

## ✨ Features
- Random motivational quote on button click
- Copy to clipboard functionality
- Minimal card UI with gradient backgrounds
- Responsive design

---

## 🚀 How to Use
1. Open the app in your browser.
2. Click **New Quote** to get inspired.
3. Click **Copy** to save the quote to your clipboard.

---

## 🎯 Example
- Click **New Quote** → “The future depends on what you do today.” – Mahatma Gandhi  
- Click **Copy** → Quote saved to clipboard.
