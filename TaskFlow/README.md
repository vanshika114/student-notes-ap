# 🚀 TaskFlow - Smart Task Manager

TaskFlow is a modern and responsive Task Management Web Application built using **HTML, CSS, and JavaScript**. It helps users organize their daily tasks, track productivity, and stay focused.

---

## 📸 Preview

TaskFlow provides:

- ✅ Add Tasks
- ✅ Edit Tasks
- ✅ Delete Tasks
- ✅ Mark Tasks as Complete
- ✅ Search Tasks
- ✅ Filter Tasks
- ✅ Progress Tracking
- ✅ Dark / Light Theme
- ✅ Local Storage Support
- ✅ Responsive Design

---

## 🌟 Features

### Task Management
- Create new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed

### Productivity Dashboard
- Total Tasks Counter
- Completed Tasks Counter
- Pending Tasks Counter
- Progress Percentage

### Search & Filter
- Search tasks instantly
- Filter by:
  - All Tasks
  - Completed Tasks
  - Pending Tasks

### Priority Management
- High Priority
- Medium Priority
- Low Priority

### Due Dates
- Assign deadlines to tasks

### Theme Support
- Dark Mode
- Light Mode

### Data Persistence
- Tasks are saved using Local Storage
- Data remains after page refresh

## 💡 How It Works

### Adding a Task

1. Enter task title
2. Select priority
3. Choose due date
4. Click Add Task

### Completing a Task

- Click the checkbox beside a task.

### Editing a Task

- Click the Edit button.

### Deleting a Task

- Click the Delete button.

---

## 📊 Productivity Tracking

TaskFlow automatically calculates:

```text
Progress % =
Completed Tasks / Total Tasks × 100
```

Example:

```text
Total Tasks: 10
Completed: 7

Progress: 70%
```
