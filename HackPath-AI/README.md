# 🛡️ FraudShield AI

FraudShield AI is a cybersecurity-inspired fraud detection dashboard that analyzes transaction patterns and identifies suspicious activities such as mule accounts, unusual transfers, and potential fraud risks.

## 🚀 Features

- Fraud Risk Score Analysis
- Suspicious Activity Detection
- Mule Account Identification
- Risk Classification (Low, Medium, High)
- Security Recommendations
- Interactive Dashboard

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript

## 📊 Example Output

- Risk Score: 88/100
- Threat Level: High
- Detected Risks:
  - Mule Account Activity
  - Rapid Fund Transfers
  - Unusual Transaction Patterns

## 🎯 Use Cases

- Banking Fraud Awareness
- Cybersecurity Demonstrations
- Fraud Monitoring Simulations
- Educational Projects
