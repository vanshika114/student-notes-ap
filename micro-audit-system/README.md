# Micro Audit System

A lightweight and modern audit management web application built using HTML, CSS, and JavaScript.

## Features

* Add audit checkpoints/tasks
* Mark audits as Pass or Fail
* Delete audit items
* Color-coded due date and status indicators
* Dark/Light mode support
* Responsive UI
* Data persistence using localStorage

## Tech Stack

* HTML
* CSS
* JavaScript

## Folder Structure

```bash
micro-audit-system/
├── index.html
├── style.css
├── script.js
└── README.md
```

## Purpose

This project was created as part of the **NSOC'26 — 20 Days 20 Projects Challenge** to provide a beginner-friendly and modern audit tracking interface.
