# Aura Study Ecosystem 🎓🚀

A premium, feature-rich, all-in-one student productivity ecosystem designed to help students manage study schedules, learning goals, assignments, and wellbeing.

## 🌟 Features

1. **Gamified Analytics Dashboard**:
   - Level up system with study XP points and streaks.
   - Live visual charts tracking weekly study time.
2. **Weekly Study Scheduler**:
   - Interactive weekly planner to map out classes, study slots, and exams.
3. **Learning Goals & Milestones**:
   - Goal creator with progress rings.
   - Milestone checklist updates goals progress live.
4. **Kanban Assignment Board**:
   - Track assignments through *To Do*, *In Progress*, *Review*, and *Completed* stages.
5. **Wellbeing & Stress Pacer**:
   - Guided deep breathing tool with inhale/hold/exhale cues.
   - Web Audio API synthesizer for custom ambient sounds (White Noise, Ocean, Binaural Beats).
6. **LMS simulated integration**:
   - Mock LMS sync button displaying academic progress and grade breakdown.
7. **Simulated AI Study Assistant**:
   - Study chatbot buddy returning mock explanations and tips.

## 🛠 Setup

To run this project locally, simply open `index.html` in any browser or launch a local dev server:
```bash
npm run dev
```
Navigate to `/student-productivity-ecosystem/index.html` in your browser.
