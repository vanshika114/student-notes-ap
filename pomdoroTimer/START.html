🍅 Production-Grade Pomodoro Timer

A professional timer that works reliably in production environments, with Web Workers, delta-time accuracy, and localStorage persistence.

🚀 Quick Start

Option 1: Open in Browser (Easiest)

  1. Open index.html directly in your browser
  2. Click "Start" to begin the 25-minute work session
  3. Timer continues accurately even if you switch tabs

Option 2: Node.js Server (Recommended)

node server.js
# Then visit: http://localhost:8000

✨ Key Features

✅ Delta-Time Architecture
No drift - calculates remaining time from target timestamp, not counter decrements.
✅ Web Worker Thread
Tick loop runs in separate thread, immune to browser tab throttling.
✅ localStorage Persistence
Auto-resume if page refreshes during active session.
✅ Mobile Audio Support
Silent unlock tone on first tap enables notifications.
✅ Clock Drift Detection
Validates time changes to catch system clock tampering.
✅ Page Title Updates
Monitor progress in browser tab: ▶️ 24:12 | Work
✅ Customizable Durations
Change work/break times in Settings (persisted).
✅ State Machine
Clean phase transitions: WORK → SHORT_BREAK → LONG_BREAK

🧪 How to Test Production Features

1. Tab Switching (Most Important Test)

  1. Start the timer
  2. Note the remaining time
  3. Switch to another browser tab for 10 seconds
  4. Switch back - timer should have counted down ~10 seconds
  5. ✅ This proves Web Worker is working!

2. Page Refresh (Persistence Test)

  1. Start timer
  2. Let it run for 5 seconds
  3. Press F5 to refresh the page
  4. Timer resumes at ~24:55 (seamlessly)
  5. ✅ Session survived the refresh!

3. Accuracy Test (Delta-Time Validation)

  1. Open DevTools (F12)
  2. Run heavy computation: 
    for(let i=0; i<1000000000; i++) x=Math.sin(i);
  3. Timer continues counting down smoothly
  4. ✅ No drift despite main thread being blocked!

See TESTING.md for complete test suite with 12+ scenarios.

⚙️ Customization

  1. Click "Settings" dropdown in the app
  2. Adjust Work Duration, Break durations
  3. Click "Apply Settings"
  4. Settings persist to localStorage automatically

For development changes, edit engine.js:

this.config = {
  workDuration: 25 * 60,        // seconds
  shortBreakDuration: 5 * 60,   // seconds
  longBreakDuration: 15 * 60    // seconds
};
