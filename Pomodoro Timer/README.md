# ⏳ Pomodoro Timer

A simple and effective **Pomodoro Timer** built with **HTML + CSS + JavaScript**.  
Stay productive with 25‑minute work sessions followed by 5‑minute breaks, complete with audio notifications and session tracking.

---

## ✨ Features
- ⏱️ 25‑minute work + 5‑minute break cycles
- 🔔 Audio notification when session ends
- 🏆 Session counter with localStorage persistence
- 🎨 Modern responsive UI
- ▶️ Start, ⏸️ Pause, 🔄 Reset controls
