# 🔢 Binary / Hex / Octal Converter

A modern **Bit Converter Tool** built with **HTML + CSS + JavaScript**.  
Convert numbers in real-time between Decimal, Binary, Hexadecimal, and Octal with a polished UI.

- 🧮 Supports Decimal ↔ Binary ↔ Hex ↔ Octal
- 🎨 Clean card-style responsive UI
- ⚡ Instant updates across all fields
