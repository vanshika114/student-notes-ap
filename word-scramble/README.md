# LEXIS — Word Scramble

A warm editorial-styled word unscrambling game with tile interaction, a countdown timer, hints, and score grading.

## How to Play
1. Click letter tiles to build your answer in the slots below
2. Click a filled slot to return the letter
3. Submit when all slots are filled (or press Enter)
4. 15 seconds per word — faster = more bonus points

## Scoring
- Correct answer: **+10 pts** (or +5 if hint used)
- Time bonus: `+floor(secondsLeft / 3)` pts
- 10 rounds total

## Controls
- **Click tiles** — place letters into answer slots
- **Click slots** — return letters to the pool
- **Enter** — submit answer
- **Backspace** — remove last placed letter
- **💡 Hint** — reveals the definition (halves points)
- **⟳ Shuffle** — randomises tile order

## Run
Open `index.html` — no setup needed.
